MOY

Welcome to Moy.Design!

We don't like the Web as it is today. It's heavy, overloaded with scripts, images, counters, social buttons, web fonts, etc. It's slow. It's clumsy. We think it can be better.

This plugin changes the appearance of websites. It serves a number of purposes:

  • Provide a way to see information from different websites in a unified way, which is customizable by end users.
  • Make web pages lighter, with less scripts, images, styling and other heavy and often unnecessary stuff.
  • Make web pages more readable, human-friendly, eye-friendly.
  • Make web pages mobile-friendly.

Check it

Below are examples of how Moy transforms web pages. You just

  1. Click a link.
  2. Enjoy an alternative look of the page.
  3. (Optional) Click the plugin icon () near the address bar to check other looks.
